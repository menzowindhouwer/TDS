The TDS IPA Console

To enter IPA symbols, use the TDS IPA Console. The Console is a stand-alone keyboard application developed by the TDS project for this purpose; it can also be used with other applications or documents.

  1. To use the Console, you should first install it.
  2. If the Console is already installed, click here to activate it.

The TDS cannot tell whether you have already installed the IPA Console. Once you have installed it, you should go to the Settings tab of the TDS interface, and change the "IPA Console launcher" setting so that you do not see this page any more.
